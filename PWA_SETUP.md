You need to create the following icon files in the `/public` directory:

## ğŸ”§ Customization Options

### Update Caching Strategies

Edit `vite.config.ts` and modify the `workbox.runtimeCaching` array:

```typescript
{
  urlPattern: /your-api-pattern/,
  handler: "NetworkFirst", // or CacheFirst, StaleWhileRevalidate
  options: {
    cacheName: "your-cache-name",
    expiration: {
      maxEntries: 50,
      maxAgeSeconds: 3600
    }
  }
}
```

### Update App Metadata

Edit the `manifest` object in `vite.config.ts` to change:
- App name and description
- Theme colors
- Display mode
- Orientation preferences

## ğŸ†˜ Troubleshooting

**Service Worker not registering?**
- Check browser console for errors
- Ensure HTTPS is enabled
- Clear browser cache and reload

**Install prompt not showing?**
- Wait ~30 seconds after page load
- Check manifest in DevTools
- Verify all PWA criteria are met

**Offline mode not working?**
- Check Service Worker status in DevTools
- Verify caching strategies in Network tab
- Ensure service worker is activated
